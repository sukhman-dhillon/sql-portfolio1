# Project 03: Customer Retention and Purchase Behavior

## Goal
Analyze customer retention and repeat purchase behavior to understand how often customers return, how valuable repeat customers are, and which segments drive long-term revenue.

## Dataset
BigQuery public dataset: bigquery-public-data.thelook_ecommerce

## Tables used
- orders
- order_items
- users

## Skills shown
- Customer-level aggregation
- Window functions
- Date difference calculations
- Cohort-style analysis
- CASE WHEN segmentation
- Filtering and grouping

## Questions answered
- How many customers are one-time vs repeat buyers?
- What share of revenue comes from repeat customers?
- How long does it take customers to place a second order?
- How does AOV differ between first-time and repeat orders?
- Which customers generate the highest lifetime value?
- How does purchase frequency vary across customers?
